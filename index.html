Music Map

You can tab through this message.

Press escape to close this message and go to the main menu.

To disable sounds, uncheck the "Sounds" button.

This map uses non-visual navigation, and is designed for use with screen readers. It shows 2000 global features from a water quality database, and starts centered on the Iberian Peninsula over Madrid.

The arrow keys move the map. Z and X keys zoom in and out. C key will center and zoom in on a selected feature.

The number keys also set zoom levels 0 through 9. Zoom 0 shows the whole world, zoom 6 is roughly the size of a country, zoom 12 is city level, and zoom 16 shows individual streets.

There is a menu with five options. Pressing enter on each will toggle the selected mode.

1. Feature selection. In this mode, press Tab and Shift-tab to move forward and back through onscreen features. Press escape to exit selection mode and return to the menu. Press Enter to examine a selected feature's data.

2. Inspect feature. In this mode, press Tab and Shift-tab to move forward and back through a selected feature's attributes. Press escape to leave the feature and return to selection mode.

3. Verbose. When navigating the map, this mode will announce zoom levels and a geolocation result for the part of the map in the center of the screen.

4. Sound. This mode plays musical notes when the map moves. The features currently onscreen are converted to a musical score from west to east, with relative vertical screen position converted to pitch. The "s" key will also trigger sounds. When this mode is active, individual features will be played in Feature selection mode.

5. Help. This button shows this help text. Press escape to close it and return to the menu.
